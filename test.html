1. View Your Portfolio

Open index.html in your browser to see your portfolio in action!

2. Setup Email (Optional but Recommended)

To enable the contact form to send you emails:

Option A: EmailJS (Easier)

  1. Go to EmailJS.com
  2. Create a free account
  3. Connect your Gmail account
  4. Create an email template
  5. Get your credentials and update script.js

Option B: Firebase

  1. Go to Firebase Console
  2. Create a new project
  3. Set up Firestore Database
  4. Get your config and update script.js

3. Customize Your Portfolio

  • Replace profile image in hero section
  • Update your contact information (email, WhatsApp, LinkedIn, GitHub)
  • Add your CV file (cv.pdf)
  • Update skill descriptions and percentages
  • Add your project images and descriptions

4. Deploy Online

Choose one of these free hosting options:

  • Netlify - Push to GitHub, auto-deploy
  • Vercel - Similar to Netlify, very fast
  • GitHub Pages - Free hosting directly from GitHub
